## JetBrains Mono

- **Files:** `public/fonts/JetBrainsMono-Variable.woff2`,
  `public/fonts/JetBrainsMono-Italic-Variable.woff2`
- **Source:** <https://github.com/JetBrains/JetBrainsMono> (release v2.304)
- **Licence:** SIL Open Font License v1.1
  (<https://github.com/JetBrains/JetBrainsMono/blob/master/OFL.txt>)
- **Note:** the upstream release ships only the variable axis as TTF.
  The WOFF2 files in this repo are produced by running `woff2_compress`
  on `JetBrainsMono[wght].ttf` and `JetBrainsMono-Italic[wght].ttf`. No
  glyph data was modified; only the container changed.
